To install, run these commands in a terminal.
This software requires gcc, libfftw3f, libsamplerate, libsndfile and boost.random. Runtime requirements are ffmpeg and python.

$ tar xjf shenidam-VERSION.tar.bz2
$ cd shenidam-VERSION/
$ mkdir build
$ cd build
$ cmake ..
$ make
$ sudo make install
